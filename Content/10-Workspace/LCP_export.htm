Exporting Raster Data from

Landscape data and Landscape Fire Behavior Model outputs can be downloaded as a Geo Tiff (.tif) raster dataset for use in other applications such as ArcMap. To download a raster dataset from :

 

  1. From your local machine, you may locate and copy the zip file, extract the contents, and store where needed.

  2. Once extracted, the file will be a GeoTif (.tif) with separate bands for each landscape feature or model output. (See the Topic on Opening rasters in ArcMap for more details if needed)

 

The GeoTiff landscape file download includes 12 bands. There are 4 in addition to the 8 that are visible within the IFTDSS application. The downloads also have different units than IFTDSS presents.

  • For instructions on how to reclass outputs such as flame length in ArcMap, see the Reclassing GeoTiff Features in ArcMap Topic.
  • For instructions on using the exported layers in FlamMap, see the help Topic on Loading GeoTiff Files into FlamMap 5.0.
