Importing Layer Symbology into ArcMap for Landscapes

Lyr files define the symbology for the layers visible in Map Studio, and are useful in setting the color ramp in ArcMap to match those in IFTDSS. For Landscapes .lyr files exist for include:

  • Elevation
  • Slope
  • Aspect
  • Fuel Model
  • Canopy Cover
  • Stand Height
  • Canopy Base Height
  • Canopy Bulk Density

We have not included the files to map the Map Zone, Existing Vegetation Type, Existing Vegetation Height, and Existing Vegetation Cover because these are not viewable in Map Studio.

The Landscape layers include both categorical and continuous data, instructions for each of these are outlined below. In addition, Elevation is dynamic based on the values present in the landscape.

Import Symbology for Fuel Model (categorical data)

The Fuel Model layer in the Landscape GeoTiff is categorical data, the following outline how to import the symbology.

  1. Open an ArcMap session and add the layers you would like to map. For more information on how do add layers to ArcMap please see the Opening an IFTDSS Geo Tiff file in ArcMap help topic.

  2. Right-click on the Fuel Model layer and click Properties.

  3. In the Layer Properties window go to the Symbology tab and click on "Unique Values". You may get a pop-up window asking if you would like to compute unique values. Click "Yes".

  4. Click on the Folder icon toward the top right. The Import Symbology window will open, click on the Folder icon and navigate to the "Fuel Models" layer file in your directory. Click Add.

  5. Click "OK" to close the Import Symbology Layer, then click "OK" to close the Layer Properties window.

  6. Now you can see Fuel Model in ArcMap the same as it appears in IFTDSS Map Studio.

Importing Symbology for Slope, Aspect, Canopy Cover, Stand Height, Canopy Base Height, and Canopy Bulk Density (continuous data)

The same steps are completed to import the symbology for Slope, Aspect, Canopy Cover, Stand Height, Canopy Base Height, and Canopy Bulk Density. The example below will use Canopy Base Height.

  1. Open an ArcMap session and add the layers you would like to map. For more information on how do add layers to ArcMap please see the Opening an IFTDSS Geo Tiff file in ArcMap help topic.

  2. Right-click on the Canopy Base Height layer and click Properties.

  3. In the Layer Properties window go to the Symbology tab and click on Classified. Then click on the Folder icon toward the top right. The Import Symbology window will open, click on the Folder icon.

  4. Navigate to the Canopy Base Height layer file in your directory. Click Add.

  5. Click OK to close out the Import Symbology window. Click OK to close the Layer Properties window. Now you can see Canopy Base Height in ArcMap the same as it is in Map Studio in IFTDSS.

The classification is completed to match the units in Map Studio, however the units in the GeoTiff are not always the same. For example, Canopy Base Height is m *10 in the GeoTiff, and m in the legend. In the image below, the value for the pixel at the red dot is 5. Which is in the >0-0.5 class in the legend (dark green). Because the data is m * 10, it represents 0.5 m in the legend.
