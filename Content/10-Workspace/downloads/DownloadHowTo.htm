Downloading Files from

  1. You may export from file lists in My Workspace or My Modeling Playground.
  2. Select the file you would like to download.

  3. Click Download.

  4. A dialogue box will appear in which you may rename the file if desired.
  5. When ready, click Download.

  6. A confirmation box will appear verifying the raster has been downloaded to your local machine in .zip file format, at this time you may also rename the file if needed. The file will be saved in your browsers "Downloads" folder, or whichever folder you have specified for your browser to use.
