Importing FlamMap input file into FlamMap 6.0 Desktop - Landscape Fire Behavior

  1. Open FlamMap 6.
  2. Open a new FlamMap project by navigating to File and clicking New.

  3. Open the Landscape file for the modeling by navigating to Landscape and clicking Open Landscape.

  4. Navigate to the folder with the landscape file, select it, then click Open.

  5. Import the IFTDSS FlamMap6 input file by navigating to Analysis Area and clicking Import.

  6. Change the file type to “WFDSS input files (*.input)” from the drop-down menu.

  7. Navigate to the folder with the input file, select it, then click Open.

  8. The Run Window will open, the first three tabs are populated with the inputs and outputs that match IFTDSS.

    Below are the inputs as displayed in IFTDSS, these are used as inputs to run Landscape Fire Behavior (FlamMap6 Basic).

    The FlamMap Inputs tab, shown below, is populated with the same wind, crown fire, and fuel moisture inputs used in IFTDSS.

    The FlamMap Fire Behavior Options tab is populated with same outputs requested by IFTDSS.

  9. Click Launch Basic FB to recreate the run from IFTDSS.
