Importing FlamMap input file into FlamMap 6.0 Desktop - Landscape Burn Probability (LBP)

You will need both the Landscape download, for the landscape GeoTIFF, and the LBP Model output download, for the modeling inputs.

Conditional Flame Length and Integrated Hazard are not available outputs from FlamMap6. Conditional Flame Length is derived from the Probability of Flame Length (FLP - 6 Bin English) outputs. Integrated Hazard is a crosswalk of Burn Probability and Conditional Flame Length classes. Those would need to be created in a GIS for runs completed solely in FlamMap6.

  1. Open FlamMap 6.
  2. Open a new FlamMap project by navigating to File and clicking New.

  3. Open the Landscape file for the modeling by navigating to Landscape and clicking Open Landscape.

    If the landscape was resampled, be sure to use the one with the correct resolution!

  4. Navigate to the landscape folder with the landscape file, select it, then click Open.

  5. Import the FlamMap6 input file by navigating to Analysis Area and clicking Import.

  6. Change the file type to “WFDSS input files (*.input)” from the drop-down menu.

  7. Navigate to the LBP model download folder that contains the input file, select it, then click Open.

  8. The Run Window opens. The first three tabs are populated with the inputs and outputs that match .

    Below are the inputs as displayed in , these are used as inputs to run Landscape Burn Probability (FlamMap6 Minimum Travel Time with Random Ignitions).

    The FlamMap Inputs tab, shown below, is populated with the same wind, crown fire, and fuel moisture inputs used in .

    In the FlamMap Fire Behavior Options tab, shown below, some of the outputs requested by are selected on this tab because they are part of the Landscape Fire Behavior (FlamMap6 Basic) family of outputs.

    The FlamMap Minimum Travel Time tab, shown below, is the portion that runs the Landscape Burn Probability model (FlamMap6 Minimum Travel Time with Random ignitions) and generates the burn probability, fire size list and probability of flame length outputs available from .

  9. Next, import the fire size list from the download package. By default, FlamMap uses the same number of ignitions that were run in , but this import is needed to replicate the ignition locations. In the Inputs tab, select the From Fire List File option and load the Fire Size List output from the download package.

  10. Click Yes to confirm your file selection.

  11. Now that your inputs and Fire List File are imported, click Launch MTT to recreate the run from .
