This section provides a series of thumbnail images and comparison table to show output differences at-a-glance.​​

All model outputs, aside from the first one indicated with a icon, may be moved up or down to change the order in which the selections are displayed. A model output may be removed from this list using the icon.​

You may toggle to a full Map Studio view at any point using the Add to map button in the top left or the View on Map Button. From Map Studio, return to the compare interface using the button in the top right.​

Easily navigate to other parts of . The button under each thumbnail image will open a copy of that model run, allowing you to make changes based on the run and save the altered run under a new name.​​
