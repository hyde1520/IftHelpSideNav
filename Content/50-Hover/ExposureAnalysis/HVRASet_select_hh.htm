Select one of the existing HVRA Sets from the drop-down of choices that equal or exceed the extent of your selected LBP.

HVRA Sets that are larger than the Landscape Burn Probability (LBP) model output will be used to create a new smaller HVRA Set that matches the LBP model output. HVRA Sets smaller than the LBP can’t be used and are not available.

If you do not have an HVRA Set you may use the button to navigate to that part of and create one.

Need a tutorial on creating an HVRA Set? See the HVRA Set step by step tutorial topic.
