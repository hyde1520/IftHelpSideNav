Primary HVRA Categories are used to label and organize your Sub-HVRAs. Sub-HVRAs are the actual shapes.

Choose your Primary HVRA Category using drop-down menu, or create your own with the Create New button.

When you select a Primary HVRA Category, will automatically create a list of the Sub-HVRAs that intersect it. You may use the Reference Data button or User Data buttons to filter the options.

Reference Data - Predefined layers in .

  • Indicates the Sub-HVRA is derived from Reference Layers
  • Indicates the Sub-HVRA is derived from National HVRA data.

User Data - Your own data, including shapes you've created in and shapefiles you've uploaded to .
