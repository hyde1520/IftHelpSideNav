The Planning Cycle - Implementation Planning

The Implementation Planning stage of the Planning Cycle allows you to:

  • Create a Landscape
  • Edit or Correct a Landscape
  • Calculate Landscape Fire Behavior
  • Compare Weather
  • Prescribed Fire Planning Support - Coming Later
  • Smoke Management Planning - Coming Later

The development team is working behind the scenes to build out the additional Implementation Planning capabilities. These functions will become available in a future release.
