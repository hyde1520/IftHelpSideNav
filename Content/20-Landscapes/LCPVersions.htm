Editing Different LANDFIRE Versions in

In , there are two methods for editing landscapes. Some editing methods require a specific LANDFIRE version or location.

  • Default Fuels Treatment Edit Rule: Available for LANDFIRE 2012 and 2014 data in the Lower 48 states.

  • User Created Edit Rule: Available for all LANDFIRE versions and locations.

    Default Fuels Treatment Edit Rules are not yet compatible with LANDFIRE 2016. Currently, attempting to edit 2016 data with this method will generate a warning message and you will be unable to proceed. In future versions of IFTDSS New Default Fuels Treatment Edit Rules will be developed.
