Monitoring Interface

The FTEM interface walks you through the treatment monitoring process. Use the tabs to select a wildfire, select your treatments, enter monitoring data, and review and complete your monitoring. Below, we walk through each stage of this process. The following steps assume you have an FTEM account. If not, visit the FTEM Setup topic for instructions. Upon opening FTEM, you will see wildfires with a split view of Map Studio on the left, and a table view on the right.

FTEM Interface Controls

Each button and tab in the FTEM interface is labeled below:

If you only see two tabs, you have Viewer role, and must request an Editor role from your FTEM profile to access the Monitoring and Complete tabs.

By default, the interface is split, displaying a map on the left, and monitoring tabs on the right. Change the interface at anytime to show only the tabs or only the map by clicking the split screen buttons in the top of the screen just below the top navigation menu. Splitting the screen to only focus on the tabs also reveals a right hand panel displaying metadata, as shown below:

The default map layers visible include:

  • Wildfire Points: Described in more detail in Wildfire technical documentation.
  • Wildfire Polygons: Described in more detail in the Treatments and Buffering technical documentation.
  • Jurisdictional Agencies Reference Layer
  • LANDFIRE 2014 Fuel Models: The 40 fuels models is visible by default, 13 is also available.
