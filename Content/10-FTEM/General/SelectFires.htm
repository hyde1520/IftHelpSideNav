Selecting and Filtering Fires

Fires are automatically filtered by date and map extent when using FTEM.

To further select wildfires use the Attribute Table in the map portion of FTEM:

  1. Click the Attribute Table button in the bottom right of the map.
  2. Click Options and select Filter.

  3. Form a filter query to constrain the fires that appear. Start with creating a filter set so you can include a date constraint. This is needed because once you start filtering, the automatic date filter in FTEM is disabled.

    You can use the Protection Jurisdiction to filter your fires by individual forest, refuge, district etc? Need to look up the jurisdiction code for your area? Use the Jurisdiction Look-up Tool.
