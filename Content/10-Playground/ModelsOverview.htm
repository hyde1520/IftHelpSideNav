Modeling Playground

Modeling Playground is the part of IFTDSS where you run models when you don't want to use the Planning Cycle. The drop-down menu at the top is where you select a model, then your landscape, etc. The middle of the page displays a list of model outputs (these are also available in MyWorkspace). Clicking on a model output displays model metadata in the right panel, and shows a list options below the output name.

Models Available

Currently the models available in focus on Fire behavior and include:

  • Landscape Fire Behavior
  • Burn Probability for Quantitative Risk Assessment [Under Development]
  • Minimum Travel Time
  • Minimum Travel Time
  • More to come....

The IFTDSS development team is working behind the scenes to build out the modeling capabilities. These functions will become available in future releases.

Modeling Output Options

Modeling output options display under the model output name once you click on it. These options include:

  • Copy: Automatically takes you to a modeling input screen for that output with the landscape and model inputs pre-populated. You may make changes to any of these and run a new model scenario. This can be especially useful if you run an Auto-97th summary and want to quickly re-run it with slightly different values.
  • View Input: Displays a model indicating all the model inputs used in that modeling output. Also displays model run name, type, date created, resolution, and owner name.
  • Download: Download a GeoTiff raster file of the output to your local computer for use outside of .

  • Delete: Delete the model run from .

    This deletes the output throughout IFTDSS (Playground and Workspace). This choice cannot be undone.

  • View on Map: Opens the model output in .
