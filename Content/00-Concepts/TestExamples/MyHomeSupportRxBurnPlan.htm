Example Task: Plan a Prescribed Burn

Creating a prescribed burn plan is a common task for Users. There are multiple ways to initiate this process, and the below steps outline one way:

  1. Create a landscape by navigating to the Planning Cycle, Landscape Evaluation stage of the cycle, selecting the Create Landscape card, and following the steps outlined in the Landscape Creation topic.
  2. Review your Landscape in Map Studio. Bring the landscape into view using the Landscape Tools widget and evaluate it spatially using the Identify, Swipe, and Layer List widgets.
  3. Make any necessary edits to ensure the landscape matches what is on the ground as close as possible by navigating to the Planning Cycle, Landscape Evaluation stage of the cycle, and selecting the Landscape Edit card. See the help topics on how to make edits and landscape rule editing for more details.
  4. Run the Landscape File Behavior (LFB) model under various weather scenarios by navigating to My Modeling Playground, selecting the Landscape Fire Behavior option following the steps outlined in the LFB tutorial.
  5. Generate Summary Reports for the model outputs by navigating to the model output file in My Workspace and using the Request Report option.
  6. Download the full report for your records, or download specific images or charts for use in your own reports.
