Example Task: Plan a Fuels Treatment

Planning a fuels treatment is a common task for Users. There are multiple ways to initiate this process, and the below steps outline one way:

  1. Create a landscape by navigating to the Planning Cycle, Landscape Evaluation stage of the cycle, selecting the Create Landscape card, and following the steps outlined in the Landscape Creation topic.
  2. Review your Landscape in Map Studio. Bring the landscape into view using the Landscape Tools widget and evaluate it spatially using the Identify, Swipe, and Layer List widgets.
  3. Make any necessary edits to ensure the landscape matches what is on the ground as close as possible by navigating to the Planning Cycle, Landscape Evaluation stage of the cycle, and selecting the Landscape Edit card. See the help topics on how to make edits and landscape rule editing for more details.
  4. Identify and mask potential treatment areas. One method to do this is to run an Automatic 97th percentile Landscape Fire Behavior run, followed by viewing the run results in Map Studio using the Simulation Output Tools widget, then using the Create/Edit shapes widget to create the treatment area mask.
  5. Next, use the Develop Treatment Alternatives (DTA) workflow to develop various treatment alternatives and compare fire behavior under those alternatives. The DTA workflow is located in the Planning Cycle under the Strategic Planning stage.
  6. Once you've completed the DTA, reports can be downloaded from MyWorkspace.
