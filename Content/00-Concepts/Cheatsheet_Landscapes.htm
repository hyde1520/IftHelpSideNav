Getting Started: Landscapes

Introduction

Creating a landscape generates an LCP file which can be used to run several models in . Landscapes you create are stored in My Workspace and are available from drop-down menus throughout whenever a landscape is required.

Need to view outputs for a specific polygon in Map Studio? Try the Add/Remove Layers option in Layer List .

All landscapes will be created with a 4980 meter buffer around them to facilitate using models such as Landscape Burn Probability. This buffer area is not included in summary reports, nor does is count against the 3.5 million acre size limit.

How to Create a Landscape

The most common way to create a landscape is from Map Studio, using the Landscape Tools widget.
