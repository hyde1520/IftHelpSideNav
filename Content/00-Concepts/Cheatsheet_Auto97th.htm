Automatic 97th Percentile Landscape Fire Behavior (Auto97th LFB) Report

Introduction

The Automatic 97th Percentile Landscape Fire Behavior (Auto97th LFB) Report will automatically run Landscape Fire Behavior across a landscape of your choosing, using 97th percentile fire weather conditions selected from the closest Remote Automated Weather Station (RAWS).

The RAWS selected, and the fuel moisture and wind scenario for the Auto97th LFB modeling, cannot be changed. It is intended to be a starting point to understand your landscape and predicted fire behavior. You may choose to run a Landscape Fire Behavior model run with user-input values in the Modeling Playground.

How to Create an Auto97th LFB Report

Access the Landscape Summary task from the Landscape Evaluation stage of the Planning Cycle

  1. Specify an existing landscape to summarize using the drop-down menu. If you do not have a landscape yet use the Create Here or Create in Map Studio buttons to create one. Creating a landscape in Map Studio will require you to navigate back the Auto97th LFB Report interface when your landscape is created. Newly created landscapes may take a few minutes to generate. If you choose a newly created landscape that is still being created, it will have an hourglass icon displayed after the landscape name; use the Refresh button to the left of the drop-down field until the hourglass is replaced by a green checkmark , indicating your new landscape is ready.
  2. (Optional) Specify an area of interest (polygon or shapefile) within an existing landscape for summary. To run a summary across the entire landscape, leave this field blank.

  3. Click the Request a Report button.

  4. A prompt appears with the name of your landscape across the top. Click Create Report to generate the summary report.

  5. The icon indicates the summary report is being generated. You can refresh and check the status by clicking the refresh button .

  6. When finished your Auto97th LFB Report will be available by clicking on the report name in the confirmation dialogue box.

    7. Depending upon landscape size, your summary may take a few minutes to complete.

It may also be viewed from My Workspace, in the same folder as the landscape you chose to summarize, as well as listed in Modeling Playground. You may also view your report outputs in Map Studio by adding the fire behavior output via the Simulation Output Tools widget. All Auto97th model output files are easily identified by their file names which end with "- Auto97th."
