The Landscape Burn Probability (LBP) model is identical to Minimum Travel Time (MTT) Burn Probability in FlamMap 6.0, with some fixed modeling inputs (see below for details).

FlamMap MTT Background

Fixed inputs in

Many of the model inputs are controlled by the user (see the Input Overview topic for more details); however some model inputs are fixed in runs and cannot be changed by the user:

  • Spotting delay: The time between ember landing and initiation of fire spread is set to 0 minutes.
  • Spotting seed: The spotting seed is used to fix spotting distance and location between model runs. In a random number is generated for an initial run for a given landscape extent, ignition and barrier (optional input). Subsequent runs using the same trifecta would re-use the spotting seed. This was done to allow for accurate comparisons between model runs where either model inputs (excluding ignition and barrier) or landscapes (edited versions of the same extent) change. Without setting the spotting seed, differences seen in fire behavior could be due to differences in spotting.
