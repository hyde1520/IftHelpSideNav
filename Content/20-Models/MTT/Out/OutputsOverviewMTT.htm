MTT Outputs Overview

Last Updated 9-9-2020 1600

Select Minimum Travel Time (MTT) outputs are available as geospatial layers in Map Studio, displayed in IFTDSS summary reports, and available for download and use within other applications. Each output is listed below.

MTT Output descriptions.

Output

Description

Arrival Time Grid The elapsed time when the fire arrived at each node. See the Arrival Time topic for more details.
Ember Landing Locations Point shapefile including ember landing locations. See the MTT Embers topic for more details.
Ember List Comma separated value file (CSV) with launch coordinates, landing coordinates, launch time, and landing time. See the MTT Embers topic for more details.
Flow Paths Major paths identified at the user-specified Interval for Minimum Travel Paths. See the Flow Path topic for more details.
Influence Grid Logarithm of the number of nodes burning as a result of burning through that particular node. See the Influence Grid topic for more details.
MTT Fireline Intensity Grid Fireline intensity of each node burned by the fire as it follows the minimum travel time path. See the Fireline Intensity topic for more details.
MTT Ignition Grid The raster interpretation of the ignition and barrier vectors by the MTT model. See the MTT Ignitions topic for more details.
MTT Rate of Spread Spread rate of the fire as it encountered each node along the minimum travel time path. See the Rate of Spread topic for more details.
