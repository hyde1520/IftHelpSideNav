How to Run Minimum Travel Time Fire Spread (MTT)

To run Minimum Travel Time Fire Spread (MTT) from the Modeling Playground, select MTT Fire Spread from the Select Model to Run drop-down menu and click Create Run.

Choose a Landscape

Choose a landscape for your run from the drop-down list.

Populate the Model Parameters and Run

  1. Specify the ignition for your landscape from the drop-down. The drop-down includes all shapes or shapefiles (points, lines, or polygons) that intersect your landscape.

    In Split screen you may create points, lines, and polygons at anytime in the map interface and these features will automatically become available in the drop-down menus for the Ignition and Barrier inputs. This includes creating a shape, uploading a shapefile, creating a shapefile subset from the Layer List widget, or from the Attribute Table.

  2. Specify whether you want any barriers on your landscape. The drop-down includes all shapes or shapefiles (lines or polygons) that intersect your landscape.

    Barriers are useful to represent non-burnable parts of your landscape such as roads or water bodies that may not have been captured within the landscape file.

  3. Enter the simulation time, including the burn period length in hours, and the number of burn periods.

    Use this input to represent the peak fire activity per day (in hours), and the number of days (referred to as burn periods) to model. Note that the weather and fuel moisture inputs remain the same regardless of the number of burn periods.

  4. Choose to either leave the Spotting probability as the default 20% (recommended for many scenarios) or adjust it as needed.

  5. Give your run a descriptive name.

  6. (Optional) If you choose to save your inputs and complete and run at a later time, click Save. The Return button will appear and clicking it will return you to Modeling Playground. To resume inputs on this MTT run, select the run in Modeling Playground and click the Edit option.

    A maximum of five MTT runs may be submitted at any given time. If you see a pop-up notifying you that this number has been exceeded, please save any inputs you’ve entered and complete your task when one of the other runs has completed.

  7. Click Run to begin the model run or Save to save any inputs you’ve entered and complete and run the model at a later time. Once run, your model output will be available in Modeling Playground and My Workspace.
