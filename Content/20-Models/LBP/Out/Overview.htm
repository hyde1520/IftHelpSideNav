Landscape Burn Probability Outputs

LFB Outputs are relative within each landscape and CANNOT be compared across landscapes.

Conditional Flame Length is an output of the burn probability model useful for hazard and risk calculations. It represents a weighted average of the flame length in each pixel over multiple simulations rather than an absolute output from one simulation. It is therefore nearly always lower than a single model run flame length output using the same input values.

Select Landscape Burn Probability outputs are available as geospatial layers in Map Studio, displayed in IFTDSS Summary Reports, and available for download and use within other applications. Availability of each output is listed below. Please see the specific download page for details on each one.
