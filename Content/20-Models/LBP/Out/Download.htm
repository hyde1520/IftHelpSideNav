Landscape Burn Probability Outputs Available for Download

Landscape Burn Probability outputs can be downloaded for use outside of IFTDSS. This can be done from either My Workspace or My Modeling Playground by clicking on the model output file to show the file options, then clicking Download.

Downloading

When the File Download window opens, by default the folder will have the same Run Name you selected when running LBP. If you prefer to change the run name for use on your on your computer you can click on the name field and type a different name, this will not change the name in IFTDSS.

The individual files within the downloaded zip folder are prefixed with the original or modified <Run Name>. Once extracted, the folder includes:

  • Multiband GeoTiff file (<Run Name>.tif, for use ArcMap) - See table below for more details.
  • Metadata files for the GeoTiff (<Run Name>.xml and <Run Name>.ovr, for use in ArcMap)
  • Inputs file (<Run Name>_FlamMap6.input file, for use in FlamMap 6.0) - See the FlamMap Inputs topic for more details.
  • Fire Size List file (<Run Name>_FIRESIZELIST.txt, for use in Excel, NotePad or ArcMap) - See the Fire Size List topic for more details.
  • Arc .Lyr symbology file for setting colors in ArcMap that match those used in .

Layers (also known as bands) included in the GeoTiff are listed in the table below.

Proportion of Flame Lengths 8-12 ft The probability of fire burning with a flame length between >8 and 12 ft (expressed as a value between 0 and 1). More detail in the Flame Length Probability topic. 6
Proportion of Flame Lengths >12 ft The probability of fire burning with a flame length between >12 ft (expressed as a value between 0 and 1). More detail in the Flame Length Probability topic. 7
Conditional Flame Length The mean flame length of all the fires that burn a given point on the landscape. More detail in the Conditional Flame Length topic. 8
Midflame Wind Speed Wind velocity at the midflame height. Calculated using wind adjustment factors based on stand height and canopy cover and the Wind Speed. More detail in the Midflame Wind Speed topic. 9
20 Ft Wind Speed Wind velocity measured 20 feet (6 m) above the top of the tree or shrub canopy, or above the ground if canopy is absent. 10
20 Ft Wind Direction Wind direction is the direction from which the wind is blowing, specified in degrees increasing clockwise from north (which is zero degrees)m Uphill or Downhill. 11
1-hr Fuel Moisture Moisture content of 1-hour fuel (dead woody fuel 0 to ¼ inch or 0 to 6.4 mm in diameter) is the mass of water within the fuel particle expressed as a percentage of the fuel particle’s oven-dry mass. More detail in the Fuel Moisture topic. 12
10-hr Fuel Moisture Moisture content of 10-hour fuel (dead woody fuel ¼ to 1 inch or 0.6 to 2.5 cm in diameter) is the mass of water within the fuel particle expressed as a percentage of the fuel particle’s oven-dry mass. More detail in the Fuel Moisture topic. 13
100-hr Fuel Moisture Moisture content of 100-hour fuel (dead woody fuel 1 to 3 inches or 2.5 to 7.6 cm in diameter) is the mass of water within the fuel particle expressed as a percentage of the fuel particle’s oven-dry mass. More detail in the Fuel Moisture topic. 14
Integrated Hazard The combination of Burn Probability and Conditional Flame Length ranging from lowest probably/lowest conditional flame lengths to highest probability/highest conditional flame lengths. More detail in the Integrated Hazard topic. 15

 

File Type Output Brief Description
.xml XML XML file that names the bands in the GeoTiff for use with ArcMap
.ovr OCR OVR file that is used along with the GeoTiff for use with ArcMap
.txt Fire Size List File including the location and final fire size (acres) for each ignition. This can be converted to a shapefile for viewing in ArcMap. See the Fire Size List Topic for more details.
.input Input File This file includes the inputs and selected outputs needed to replicate your run in FlamMap6 desktop. See the FlamMap Input File Topic for more details.
