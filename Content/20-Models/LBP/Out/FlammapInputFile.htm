Input File for FlamMap 6.0

The Input file is only available in the download package from .

The Input File is a text file that contains information about the model run, the inputs used to run the model, and the outputs requested by the model. An Inputs file can be opened with a text editor such as NotePad, but the real utility is in FlamMap 6.0. This file can be used along with your landscape file (available for downloaded from ) and your fire size list (in the same download package as the Input file) to run the same simulation in FlamMap 6.0.
