Using the 'Input' File from LFB Downloads in FlamMap 6.0

When downloading LFB data from , you will notice a ."input" file in addition to the GeoTIFF raster files when you unzip the downloaded information. The .input file may be used outside of in the FlamMap.

.input files contain the model run inputs you entered in including:

  • 1-Hr Fuel Moisture
  • 10-Hr Fuel Moisture
  • 100-Hr Fuel Moisture
  • Herb Fuel Moisture
  • Shrub Fuel Moisture
  • Foliar Moisture content
  • Crown Fire Method
  • Wind Direction
  • Wind Speed
  • Whether gridded winds were generated (Yes/No).
  • Gridded winds resolution if gridded winds were used.
