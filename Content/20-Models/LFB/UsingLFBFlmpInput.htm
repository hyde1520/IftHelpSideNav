Using the 'Input' File from IFTDSS LFB Downloads in FlamMap

FlamMap Version 6

When downloading LFB data from IFTDSS, you will notice a ".input" file in addition to the GeoTiff raster files when you unzip the downloaded information. The .input file may be used outside of IFTDSS in the FlamMap.

.input files contain the model run inputs you entered in IFTDSS including:

  • 1-Hr Fuel Moisture
  • 10-Hr Fuel Moisture
  • 100-Hr Fuel Moisture
  • Herb Fuel Moisture
  • Shrub Fuel Moisture
  • Foliar Moisture content
  • Crown Fire Method
  • Wind Direction
  • Wind Speed
  • Whether gridded winds were generated (Yes/No).
  • Gridded winds resolution if gridded winds were used.
