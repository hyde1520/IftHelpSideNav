About Landscape Fire Behavior (LFB)

Introduction

One of the first stages in the landscape evaluation stage of landscape planning includes running a basic fire behavior model for your area. The Landscape Fire Behavior (LFB) model will create spatial outputs viewable in Map Studio as well as summary reports. You can access the LFB model from the Landscape Evaluation stage of the Planning Cycle, as well as from Modeling Playground.

The LFB model is the one that is automatically run when you create an Auto97th Landscape Summary.

Defining Landscape Fire Behavior

Landscape Fire Behavior in is driven by FlamMap. FlamMap is a fire behavior mapping and analysis software application that computes potential fire behavior characteristics (such as spread rate, flame length, and fireline intensity) over an entire landscape under constant weather and fuel moisture conditions input by the user.

Why Create a Landscape Fire Behavior Model (LFB) Run

Landscape Fire Behavior outputs can be useful in prioritizing treatment areas. Model outputs can be viewed spatially in Map Studio or as summary reports. These model outputs are also used in the Developing Treatment Alternatives task in .

Using the LFB model

To run an LFB model you will need to have created a landscape in . You can run LFB on newly created landscapes as well as ones you've made edits to. In addition to the landscape, you will provide the model with inputs describing fuel and moisture conditions.
