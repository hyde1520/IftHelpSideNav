Fire Size List

When IFTDSS includes the ability to compare landscape treatment alternatives using the Landscape Burn Probability model, the Fire Size List will be used in subsequent model runs to hold the ignition locations constant. This is important to do understand the effect of changes to fuels and forest structure on resultant burn probability and intensity (conditional flame length).

The Fire Size List file is only available in the download package from IFTDSS. It is a text file (csv or comma separated values format) that contains information about all the random ignitions in a Landscape Burn Probability run.

The file contains the following information:

  • FireNum - the fire number, a unique identifier for each ignition
  • XStart - the horizontal coordinate of the ignition point in meters
  • YStart - the vertical coordinate of the ignition point in meters
  • Acres - the final fire size in acres

The Fire Size List can be opened with a text editor program such as NotePad for viewing, opened with a spreadsheet program such as Excel for further analysis, or imported into a GIS such as ArcMap for geospatial viewing or further analysis.
