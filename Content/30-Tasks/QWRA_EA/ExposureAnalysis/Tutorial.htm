Below are step by step instructions for running and viewing Exposure Analysis, including:

  • Running Exposure Analysis
  • Viewing Results in Map Studio
  • Creating a Summary Report

Running Exposure Analysis

  1. From the Cycle, select the Strategic Planning stage and click the Exposure Analysis option under Hazard, Exposure, and Risk Assessments.

  2. Select a Landscape Burn Probability (LBP) output from the drop-down menu. If you have not yet run the LBP model, you may navigate to that part of IFTDSS via the button located to the right of the drop-down menu. See the LBP Overview topic or the LBP Tutorial topic if needed.

  3. Select an HVRA set from the drop-down menu. If you have not yet created an HVRA Set, you may navigate to that part of IFTDSS via the button to the right of the drop-down menu. See the HVRA Overview topic or the Map Values Tutorial topic if needed.

    The extent of your LBP model run will dictate which HVRA Sets are available in the next step. Only HVRA sets that fully match or exceed the LBP extent are available, as full coverage is needed for Exposure Analysis calculations.



  4. (Optional) If you select an HVRA Set with a greater extent than your selected LBP model output, you will be prompted to create a smaller clipped version. Click the link that appears in the yellow notification box, give the new HVRA Set a name, and click Save. You will not leave the Exposure Analysis interface.

    Any Sub-HVRAs that are displayed in red italics during this process are outside of the LBP model extent and will not be included in the new clipped HVRA Set.

  5. Give your Exposure Analysis a descriptive name and click Run.

Viewing an Exposure Analysis in Map Studio

To view and Exposure Analysis Output in Map Studio:

  1. Navigate to My Workspace, locate your Exposure Analysis output, and click the file to show the file options. The Exposure Analysis output is stored in the same folder as the LBP model output you used to generate it.

    You can use the Display filters in the left panel to quickly sort your Workspace content by type.

  2. Click the View on Map button.

    Exposure Analysis creates a rasterized version of your HVRA Set for quantification - this makes point and line features difficult to view. The best visual display can be had by overlaying the original HVRA Set with LBP model outputs in Map Studio.

  3. (Optional) You may constrain the Exposure Analysis Outputs to display only within an area of interest by going to Layer List, clicking the More Options button for the Exposure Analysis layer, and using the "Mask" option.

    Masked Burn Probability and Hazard values will change based on Map Extent. See the LBP Masks and Dynamic Output topic for more detail.

    Sub-HVRAs that do not intersect the newly masked area will be indicated in the Layer List with “no data in mask” appearing after the Sub-HVRA name.

[COMPLETE WHEN REPORTS ARE READY] Create and view a summary report for Exposure Analysis

To create a summary and view a summary report:

  1. Navigate to My Workspace, locate your Exposure Analysis output, and click the file to show the file options. The Exposure Analysis output is stored in the same folder as the LBP model output you used to generate it.

  2. Click the Request a Summary Report button.

  3. Locate the summary report file, in the same folder as the Exposure Analysis output, and click the View Summary button.
