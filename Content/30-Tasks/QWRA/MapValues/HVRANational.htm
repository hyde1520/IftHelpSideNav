Map Values - Reference Data

The Reference Data in Map Values is a combination of standard reference layers and National HVRA Data layers which represent those HVRAs layers used by Dillon (2020) during the National Wildfire Risk Assessment for Forest Service Lands. The Reference Data are assigned to Primary HVRA Category(s) based on what the layer represents. When a Primary HVRA Category is selected, determines available Sub-HVRAs within the specified geographic extent, including your shapes and shapefiles.

Your geographic extent may not always include every Sub-HVRA in the Primary HVRA Category.

Available reference layers in Map Values include:

  • Reference Data includes those reference layers that are national in scope and represent features that are considered values, thus not every reference layer is available. This data is indicated by the icon.
  • Highly Valued Resource or Asset (HVRA) National Data is part of the of the HVRA dataset used in the National Wildfire Risk Assessment for Forest Service Lands (NaWRA, Dillon 2020) and are national in scope including non-Forest Service lands. This data is indicated by the US map icon.

The Reference Data are listed below for Sub-HVRAs, grouped by Primary HVRA Category. Each is linked to a metadata page with more detail. We've also included Response Functions used for Quantitative Wildfire Risk Assessment.
