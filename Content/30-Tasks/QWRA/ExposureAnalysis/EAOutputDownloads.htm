You can download Exposure Analysis (EA) outputs for use outside of IFTDSS. This can be done from My Workspace by clicking on the Exposure Analysis file to show the file options, then clicking Download.

Downloading

When the File Download window opens, by default the folder has the same Exposure Analysis name you selected when running an EA. If you prefer to change the name for use on your computer you can click on the name field and type a different name, this does not change the name in .
