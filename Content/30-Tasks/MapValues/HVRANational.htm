Map Values - Reference Data

The Reference Data in Map Values is a combination of standard IFTDSS reference layers and National HVRA Data layers which represent those HVRAs layers used by Dillon (In Press) during the National Wildfire Risk Assessment for Forest Service Lands. The Reference Data are assigned to Primary HVRA Category(s) based on what the layer represents. When a Primary HVRA Category is selected, IFTDSS determines available Sub-HVRAs within the specified geographic extent, including your shapes and shapefiles.

Your geographic extent may not always include every Sub-HVRA in the Primary HVRA Category.

Available reference layers in Map Values include:

  • IFTDSS Reference Data includes those reference layers that are national in scope and represent features that are considered values, thus not every IFTDSS reference layer is available. This data is indicated by the IFTDSS icon.
  • Highly Valued Resource or Asset (HVRA) National Data is part of the of the HVRA dataset used in the National Wildfire Risk Assessment for Forest Service Lands (NaWRA, Dillon In Press) and are national in scope including non-Forest Service lands. This data is indicated by the US map icon.
