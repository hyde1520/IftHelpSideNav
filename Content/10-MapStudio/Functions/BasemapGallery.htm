Basemap Gallery

Basemap Gallery allows you to change the basemap for your area. Basemap options include:

  • Dark Gray Canvas (metadata for Dark Gray Canvas )
  • Imagery (metadata for Imagery )
  • Imagery with Labels (metadata for Imagery with Labels )
  • Light Gray Canvas (metadata for Light Gray Canvas )
  • National Geographic (metadata for National Geographic )
  • Oceans (metadata for Oceans )
  • Open Street Map (metadata for Open Street Map )
  • Streets (metadata for Streets )
  • Terrain with Labels (metadata for Terrain with Labels )
  • Topographic (metadata for Topographic )
  • USA Topo Maps (metadata for USA Topo Maps )
  • USGS National Map (metadata for USGS National Map )

Basemap Gallery Layer Creation Dates

The layers available in Basemap Gallery are typically within 3-5 years of currency. For date information on a specific layer, consult the links provided above.
