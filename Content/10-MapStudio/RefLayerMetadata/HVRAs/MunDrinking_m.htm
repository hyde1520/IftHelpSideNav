National HVRA: Infrastructure - Municipal Drinking Water

Brief summary

This dataset includes coverage for the conterminous US.

Description

Overview

Municipal Drinking Water is part of the Highly Valued Resources and Assets (HVRA) data used in the National Wildfire Risk Assessment for Forest Service Lands (NaWRA, Dillon In Press). Municipal Drinking Water can be used as sub-HVRAs within the Map Values, Exposure Analysis and Quantitative Wildfire Risk Assessment functionality within the Interagency Fuel Treatment Decision Support System (IFTDSS, https://iftdss.firenet.gov)

Data Details

The Municipal Drinking Water was derived from the USDA Forest Service Forests to Faucets project (Weidner and Todd 2012, https://www.fs.fed.us/ecosystemservices/FS_Efforts/forests2faucets.shtml ). The Surface Drinking Water Importance (IMP) index takes into account water supply, spatial flow through the landscape, and the downstream drinking water demand (Weidner and Todd 2012). The IMP index was mapped to the hydrologic unit 12 (HUC 12 scale). Non-zero IMP index values were split into 100 quantiles, so the values range from 0 to 100.

Municipal Drinking Water for each HUC 12 was assigned to one of ten classes based on the IMP index value:

  • Lowest value (< 10th percentile) - IMP index 0-9
  • Extremely low value (10th - 19th percentile) - IMP index 10-19
  • Very low value (20-29th percentile)- IMP index 20-29
  • Low value (30th - 39th percentile)- IMP index 30-39
  • Moderately low value (40th - 49th percentile)- IMP index 40-49
  • Moderately high value (50-59th percentile)- IMP index 50-59
  • High value (60th - 69th percentile)- IMP index 60-69
  • Very high value (70-79th percentile)- IMP index 70-79
  • Extremely high value (80th - 89th percentile)- IMP index 80-89
  • Highest value (90th - 99th percentile) - IMP index 90-100
