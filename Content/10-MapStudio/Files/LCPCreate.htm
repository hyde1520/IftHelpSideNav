Creating a Landscape

The first step in working with landscapes in is to create a landscape and evaluate the information. In , a landscape file refers to the spatial file that contains vegetation and fuel data in .lcp file format.

In future releases of you will be able to upload your own .lcp files

You may create a landscape using LANDFIRE data, either by using Map Studio and the Landscape Tools widget, or through the Auto97th LFB Report task in the Landscape Evaluation Stage of the Planning Cycle. Instructions for each are outlined below:

Create a new landscape from the Planning Cycle Create a new landscape in Map Studio Create a new landscape using the "Create Here" button

The "Create Here" button is an option when creating an Automatic 97th Percentile Landscape Fire Behavior (Auto97th LFB) Report. This allows you to conveniently create a new landscape on that same page without having to navigate to another part of to create a landscape. To create a landscape after clicking this button:

Once you click Create, you will be returned to the Landscape Summary task, and the field next to My Landscapes, displays the name of your newly created landscape, with an hourglass next to it, indicating it is being created. Once created, the hourglass is replaced by a green checkmark. You may have to use the refresh button next to the landscape name to see the green checkmark.
